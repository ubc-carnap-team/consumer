This is a simple demo of how we insert the tree widget.

Suppose we want to set up an exercise for a widget initialized with the initial formula set needed to prove "modus tollens". I'll put a div here which has the id "widget-one", and then in javascript, I'll run this code. Rudolf.createTree('element-one', "P->Q,~Q,~~P") Notice that the initial formula set is passed in as a string, with commas in between the formulas. The complete code needed to insert an element is, in full: 

        
          <div id="element-one">Widget one goes here.</div>
          <script>
            <code type>Rudolf.createTree('element-one', "P->Q,~Q,~~P")</code>;
          </script>

I'm going to insert that code verbatim in the html document right after the end of this sentence.
